# Date Recommend Program

Python으로 만든 데이트 추천 프로그램입니다.

## 프로젝트 소개
사용자가 지역, 예산, 데이트 종류를 선택하면
조건에 맞는 데이트 장소를 추천해주는 프로그램입니다.

## 기능
- 지역 선택 (평택, 강남, 김포)
- 예산 선택 (1만원 이하, 3만원 이하, 상관없음)
- 데이트 종류 선택 (맛집, 산책, 영화, 카페)
- 조건에 맞는 장소 추천
- 자연스러운 추천 문장 출력

## 사용 기술
- Python
- 함수
- 조건문
- 문자열 출력
- 입력 처리

## 실행 방법
```bash
python date_recommend.py
